# <<创建分类>>

Catelog->Manage Categories

添加分类很简单,有几个重要的按钮要说下:

```
Add Root Category : 主分类,一个store对应一个Root Category当你有多个store的时候,并且store之间的分类都不一样的话,就可以添加主分类。

Add Subcategory: 子分类。

```

###目标

添加一个分类category #1.并且在前台显示出来
